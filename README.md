# Analyse Multi-Plateformes — Google Ads, Meta Ads & TikTok Ads

## Objectif
Comparer les performances publicitaires sur 3 plateformes majeures 
sur 1 800 campagnes (2024) et identifier la meilleure allocation budgétaire.

## Résultats clés
- TikTok Ads : ROAS 7.62 — meilleur ROI malgré seulement 24% du budget
- Google Ads : absorbe 57% du budget pour le ROAS le plus faible (3.47)
- Réallocation optimisée : +21% de revenue soit +11M€ avec le même budget
- E-commerce & SaaS : les industries les plus rentables (ROAS > 5)

## Recommandations
- Réduire Google Ads de 57% à 30% du budget
- Augmenter TikTok Ads de 24% à 45% du budget
- Prioriser les campagnes Search et Display sur TikTok

## Stack technique
- Python · Pandas · NumPy · Matplotlib · Seaborn
- Jupyter Notebook

## Dataset
Global Ads Performance Dataset (Google, Meta, TikTok) — Kaggle
